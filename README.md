Fuckyeah project by [@Manvel](https://github.com/Manvel) and [@diegocarloslima](https://github.com/diegocarloslima).

## Installation

```
npm i
```

## Starting server

### With cache

```
npm start
```

### For development

```
npm run dev
```

## Generating static content

```
npm run build
```

## Deploying

```
npm run deploy
```
